Unix 的优点

Linux 承袭了大量 Unix 的设计精华主要包括下面这些：

  • 系统接口简单，也就是说系统调用设计的很合理

  • 一切都是文件，这样就可以把数据和设备的访问统一到 open(), read(), write(), lseek(), close() 这组系统调用中

  • 使用 C 语言编写，有非常好的可移植性（这个应该是相对与汇编语言来说吧）

  • 快速的进程创建（fork）

  • 简单而稳定的进程间通信机制（IPC）
